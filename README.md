# Project-LoopDS1-dict-numpy-python
This project about learning looping data structure (numpy) in python (In Indonesia)

Dalam looping data structure bisa menggunakan dictionary dan numpy. Dalam hal ini perulangan menggunakan for.
Key dan value dapat disingkay menjadi k dan v. 
Dalam perulangan ini harus menggunakan metode .items agar tidak terjadi error.
nditer digunakan sebagai iterator objek agar dapat berada di atas atau melewati array.
